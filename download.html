Ubuntu packages

Ubuntu packages for 12.10 (quantal), i386 and amd64. Type the following commands:

  • sudo add-apt-repository ppa:julien-rosener/digitalscratch
  • sudo apt-get update
  • sudo apt-get install digitalscratch
